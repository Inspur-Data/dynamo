# Local Benchmarking

Single-node performance testing for Dynamo.

## Quick Start

```bash
# Example benchmark command
dynamo-bench --model meta-llama/Llama-3.1-8B --batch-size 32
```

## Benchmark Scenarios

- Throughput testing
- Latency profiling
- Memory usage analysis
- GPU utilization
